🧰 Project Tools Breakdown

ImageKit.io
👉 Used for uploading, optimizing, and delivering user files (images/docs) via CDN with transformation and security options.

Next.js
👉 The React-based full-stack framework powering the frontend and backend, including routing, SSR, API routes, and UI.

Clerk (Auth)
👉 Handles user authentication (sign up, sign in, session management) with OTP/email/password login out of the box.

Neon (Postgres)
👉 A serverless PostgreSQL database used to store file metadata, folders, users’ file states (e.g., starred/deleted), etc.

Drizzle (ORM)
👉 Used to define and query database tables in a type-safe way directly in your code, replacing raw SQL.

HeroUI
👉 A Tailwind-compatible UI component library used for building responsive and elegant interfaces quickly.
